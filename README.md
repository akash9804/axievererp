# AXIEVERERP System

## Requirements
- PHP 8.x
- Laravel 10.x
- MySQL

## Setup Instructions
```bash
git clone https://github.com/akash9804/axievererp.git
cd axievererp
composer install
php artisan key:generate // if missing or not working
php artisan migrate --seed
npm install 
npm run dev
php artisan serve


for reference Database file added to root folder DB File: axiever_erp.sql
